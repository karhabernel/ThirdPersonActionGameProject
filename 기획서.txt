◎ 게임 장르
- 1인용 3인칭 액션 게임

◎ 게임 개요
- 데빌 메이 크라이, 갓 오브 워 같은 3인칭 액션 게임

◎ 주요 기능
- 콤보 점수 시스템
 · 같은 행동 반복시 콤보 점수 획득량이 점점 낮아지도록 설정
 · 콤보 중 다른 무기를 활용하여 콤보를 이어갈 시 추가 점수를 부여하도록 설정
 · 콤보 게이지를 사용하여 콤보 게이지가 전부 떨어지면 콤보가 마무리되고, 최종 점수를 재화로 획득하도록 설정
 · 콤보 게이지는 자연적으로 감소하고 콤보 수가 많을수록 더 빨리 감소하나, 콤보 점수 획득량에 비례하여 콤보 게이지를 획득하고 적에게 피격 시 콤보 게이지가 감소하도록 설정
- 콤보 시스템
 · 좌클릭(근접 무기)와 WASD 커맨드 조합, 콤보 카운트 수에 따라 다른 기술이 나가도록 설정
 · 우클릭(원거리 무기)에는 콤보는 없으나, 좌클릭을 통한 콤보 중 언제든지 원거리 무기 사용 가능 (콤보 카운트는 초기화)
- 일정 범위 내 적 자동 조준 시스템
 · Sphere Trace를 통해 주변 적 감지
- 적 스폰 시스템
 · 특정 전투 지역 진입시 설정된 적 수가 전부 죽을때까지 계속 스폰 및 지역 잠김
 · 적이 전부 죽을시 전투지역 밖으로 이동 가능
- 스테이지 진입 시 시네마틱 재생 & 카메라 뷰 변경
- (콤보중) 무기 스왑 시스템
- Data Asset을 사용한 무기 관리
- 장비(부적) 착용을 통한 스탯 변경
 · 특정 적 상대 추가데미지
 · 최대체력 추가 등
- Components를 활용한 State, Status, Option 관리
- 적 처치 및 콤보 점수에 따른 재화 획득 및 기능 해금(상점)
 · 커맨드 해금
 · 무기 해금
 · 부적 해금
- 특정 지역 체크포인트(세이브)존 존재, 세이브 시 별도의 파일을 통해 현재 상태 및 진행상황 기록되도록 설정
- 인벤토리 기능
 · 현재 가진 재화, 무기, 부적 등을 확인 및 세팅 가능
- 위젯
 · 체력 위젯 : 나와 적의 체력을 볼 수 있는 위젯
 · 무기 위젯 : 현재 선택된 무기를 실시간으로 볼 수 있는 위젯
 · 메뉴 위젯 : F10을 누르면 메뉴 팝업, 메뉴 화면에서는 기본적으로 일시 정지 상태이고, 재시작, 메인 화면 이동, 게임 종료 선택 가능
 · 인벤토리 위젯 : 인벤토리 UI 제공, 이 때 이동은 불가하고 UI만 클릭 가능하도록 설정
 · 상점 위젯 : 특정 지역에서 상점 진입 시 아이템 구매가 가능한 UI 제공, 이 때 이동은 불가하고 UI만 클릭 가능하도록 설정

◎ 조작법 : 키보드 + 마우스
○ 키보드
- 이동 : WASD
- 점프 : 스페이스 바
- 회피 : 왼쪽 Shift
- 무기 선택 : 숫자키 1, 2, 3, 4
- 특수 기능(필살기, 스킬) : F
- 타겟 변경 : Q, E
- 메뉴 : F10

○ 마우스
- 마우스 이동 : 화면 돌리기 (3인칭일 시, 특정 스테이지 뷰에서는 화면 회전 X)
- 왼쪽 클릭 : 근접 공격(근접 무기 사용)
- 오른쪽 클릭 : 원거리 공격(원거리 무기 사용)
